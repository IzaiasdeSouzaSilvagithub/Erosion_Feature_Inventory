# Erosion Feature Inventory – Miguel Calmon-BA, Brazil (2023–2026)

## Overview

This repository contains a **field-based erosion feature inventory** comprising **2,224 georeferenced points** collected during successive field campaigns between 2023 and 2026 in the municipality of Miguel Calmon, Bahia, Brazil, within the semi-arid region of the country. The study area is located in the Riacho da Consulta watershed, Brejo Grande district, encompassing the Caatinga biogeographic domain.

Each point represents an observed linear erosion feature classified *in situ* into one of three categories:

| Erosion type (Portuguese) | Erosion type (English) | Count | Percentage |
|---------------------------|------------------------|-------|-------------|
| Boçoroca | Gully | 749 | 33.7% |
| Ravina | Ravine | 673 | 30.2% |
| Sulco | Rill | 802 | 36.1% |
| **Total** | | **2,224** | **100%** |

Classification criteria follow the standard definitions in the geomorphological literature [1,2]:
- **Rill (Sulco)** : small ephemeral channels with cross-section < 0.3 m, removable by conventional tillage
- **Ravine (Ravina)** : incised channels with depth between 0.3 and 1.0 m, not removable by tillage
- **Gully (Boçoroca)** : permanent channels with depth > 1.0 m, presence of a steep headcut, and continuous lateral expansion

## Coordinate Reference System

All coordinates are provided in **UTM zone 24S** (Southern Hemisphere), datum **SIRGAS 2000**, corresponding to **EPSG:31984**. This is the official geodetic reference system for Brazil and is recommended for large-scale mapping projects in the region.

|-----------|-------|

| False easting | 500,000 m |
